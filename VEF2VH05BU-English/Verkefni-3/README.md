# **SVG Vector Graphics - _Scalable Vector Graphics_**

Vector graphics are an **XML vector image format** for defining two-dimensional graphics. Vector drawings are used in all fonts on your computer, and most of the icons you see on your phone are vector graphics.

![SVG logo](Námsefni-3/SVG-symbol/Logo-200.svg)

Create a simple **SVG company logo**, and insert it into a website. The logo should be located at the top of the website. Let the logo scale according to **breakpoints** in the stylesheet.

The logo can be drawn in the [InkScape](https://inkscape.org/release/inkscape-1.2.2/) vector drawing program.

## **Icomoon Icon Font**

An **icon font** is a collection of icons that has been placed in a font and can be used in the same way as other fonts in a browser. The main reasons for using icon fonts are that you can easily change their **size, color, and shape** using `font-size:` and `color:`.

Put together your own **icon font**. This can be done on the [Icomoon.io](https://icomoon.io/) website. After importing the icons into your project, they can be treated like a normal **font-family** on the website. Use the icons in the table of contents (navigation) and the `<footer>` social media links.

*   Download the icon font by clicking **"Generate Font"**.
*   The folder that Icomoon downloads to your computer contains a **"style.css"** and a **"font"** folder with the icon fonts you selected. Copy the "style.css" and the "font" folder.
*   The **Demo.html** page shows you how to place the icons on a website.
*   The icons can be handled like a normal **font-family** on the website.

_Choose icons that match your table of contents._
