4. Detailed Project Descriptions

* Project 1: Students set up a web framework (web framework).
  * Includes color theme, grid, tables, and forms.
* Project 2: Table of contents is placed in a drop-down list and a pop-up window is created.
* Project 3: Work with SVG drawings and an icon font library is created.
* Project 4: Animation using CSS transition, transform, and @keyframe.
* Project 5: Web for an event.
  * Content is optional; students must:
    * Gather text and images for use.
      * Determine the color theme and font selection (Style-tile).
      * Create a site-map.
      * Create a navigation system (Wireframe).
* Project 6: Web for an event.
  * Production and finishing.
  * Website is published at https://(user).github.io/(subdomain).

Editor's Note: Please be advised of data inconsistencies in the source material. The Study Plan (Section 2) lists Project 5 at 5% and Project 6 at 25%, while the Assessment section (Section 5) lists them as 10% and 20% respectively. Additionally, the submission deadline of 04/03 in Section 3 is logically inconsistent with the Spring 2026 Span 2 schedule. Students should verify these details with their instructor.
