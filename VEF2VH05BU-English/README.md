### Assignment Description

1. Assignment 1. Students set up a web framework.
   - Color theme, grid, tables, and forms.
1. Assignment 2. Place the table of contents in a drop-down menu and create a pop-up window.
1. Assignment 3. Work with SVG graphics and create an icon font set.
1. Assignment 4. Animation with CSS _transition, transform & @keyframe_.
1. Assignment 5. Event website.
   - Website content is optional, but students must:
   - collect text and images that can be used
   - decide on the website color theme and font choices (_Style-tile_)
   - create a site map (_Site-map_)
   - create a navigation wireframe (_Wire-frame_)
1. Assignment 6. Event website.
   - production and finalization
   - publish the website at https://(user).github.io/(subdomain)
