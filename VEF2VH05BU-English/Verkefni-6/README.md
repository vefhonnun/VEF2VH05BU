# Assignment 6

# Website for an Event

### Goals:
Students should be able to work in an organized way when building a website.

### Design is developed according to the preparation and planning materials from Assignment 5

What should be included on the website:

  * Information about the event
  * Timing/date
  * Schedule
  * Event registration (_form appearance only - no functionality_)
  * Information about the company hosting the event
  * Animation to capture the user's attention

> The website should reflect the practical skills you have gained in this course as well as the previous course, VEFTH1.

### Assessment 25% - _Key Assessment Component_

- **7%** Content
    * SVG logo is in the &lt;header>
    * Images and text
    * Form with required input fields
    * Table visible on all screen sizes
    * Navigation - drop-down menu on mobile
- **7%** Layout and visual design
    * Pico CSS framework
    * Two-part color scheme (_Light & dark_)
    * Font selection
    * Responsive design, images adapt to different screen sizes
- **5%** Animation
    * Animation
    * Slideshow, if applicable
- **3%** Finishing
    * One compressed stylesheet linked to HTML pages
    * All images in a separate folder
    * HTML5 and CSS syntax is correct
- **3%** Website published on the internet
    * https://(_user_).github.io/(_subdomain_)

#### Assignment Submission

- All work materials must be submitted in Assignment 6 in Inna.
- A link to the website https://(_user_).github.io/(_subdomain_) should be entered in _comment to teacher_.
